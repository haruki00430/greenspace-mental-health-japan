## 1. プロジェクト概要

### 研究タイトル
**「緑地環境と精神科処方薬の地域分布に関する空間疫学研究：二次医療圏単位の横断的分析」**

### 英文タイトル（案）
**"The Urban Greenspace Paradox: An Ecological and Spatial Analysis of Natural Environments and Psychotropic Drug Prescriptions in Japan"**

### 研究の独自性
- 都市緑地において精神科処方薬が増加するという「Urban Greenspace Paradox（都市緑地の逆説）」の検証
- 国土地理院の土地利用データ（緑地率：公園vs森林の層別化）とNDB処方薬データを統合した初の全国規模研究
- 二次医療圏（N=335）という詳細な空間単位での解析
- 空間ダービンモデル（SDM）を用いた空間的波及効果（Spillover effect）および環境要因の厳密な評価
- 既存プロジェクト（NDB_XXX_slope_fracture、NDB_AF_Anticoagulation_Spatial）の成功パターンを応用

## 2. 研究背景・意義

### 2.1 背景

#### 精神疾患の社会的負担
- 日本における精神疾患患者数は約419万人（2017年厚労省患者調査）
- うつ病・不安障害の有病率上昇により、抗うつ薬・抗不安薬の処方量が増加
- 精神疾患は医療費だけでなく、労働生産性の低下など社会経済的損失が大きい

#### 緑地環境と精神保健のエビデンス
- 海外研究では、緑地へのアクセスが精神的健康（well-being）と正相関
- 緑地は運動促進、社会的交流、ストレス緩和などの経路で精神保健に寄与
- 日本では都市部と地方で緑地率に大きな格差が存在

#### 空間疫学の必要性
- 精神保健は個人要因だけでなく、地域環境（built environment）の影響を受ける
- 従来の研究は都道府県単位が多く、二次医療圏レベルの詳細な空間分析は不足
- 空間的自己相関（隣接地域の影響）を考慮した統計手法が必要

### 2.2 研究の意義

#### 学術的意義
1. NDBオープンデータと地理情報を統合した新規研究領域の開拓
2. 空間統計手法による因果推論の精緻化
3. 環境疫学・予防医学の理論的枠組みへの貢献

#### 政策的意義
1. 都市計画（緑地整備）の優先順位設定へのエビデンス提供
2. 精神保健対策における環境介入の有効性を示唆
3. 地域格差是正のための資源配分の最適化

## 3. 研究目的・仮説

### 3.1 研究目的

### 3.1 研究目的

#### 主要目的
二次医療圏別の緑地率（公園・森林等）と精神科処方薬（抗うつ薬・抗不安薬・睡眠薬）の処方動向（医療利用）との関連を、医療提供体制・社会経済要因・気象要因を調整した上で空間統計手法を用いて評価し、「Urban Greenspace Paradox」の存在を検証する。

#### 副次目的
1. 精神科処方薬の空間的分布・偏在の可視化
2. 緑地率と処方薬の関連における空間的波及効果（Spillover effect）の検証
3. 医療提供体制（精神科診療所密度、医師数）および社会経済要因（失業率、所得、高齢化率、単身世帯割合）の交絡・修飾作用の評価
4. 日照時間（季節性うつとの関連）の交絡因子としての影響評価

### 3.2 研究仮説

#### 主仮説
**「緑地率が高い二次医療圏では、医療提供体制および社会経済要因の交絡を調整した後においても、精神科処方薬の処方動向に逆説的な増加（あるいは質のによる異質性）が観察される（The Urban Greenspace Paradox）」**

#### 副次仮説
1. 精神科処方薬の分布には正の空間的自己相関が存在する（Moran's I > 0, p < 0.05）
2. 緑地の性質（整備された都市公園 vs 未整備の森林）によって処方動向への影響が異なる
3. 隣接圏域の緑地や医療資源の豊富さが、当該圏域の処方量に波及効果をもたらす（SDMによる検証）
3. 日照時間は独立して処方薬と負相関する

## 6. 統計手法の詳細

### 6.1 年齢標準化

#### 直接法（Direct Method）
```
Age-standardized rate = Σ(age-specific rate_i × standard population_i) / Σ(standard population_i)
```
- 基準人口: 2020年二次医療圏別人口（5歳階級）
- NDB提供の年齢階級: 0-4, 5-9, ..., 80-84, 85+（18階級）

### 6.2 空間重み行列（Spatial Weights Matrix）

#### Queen Contiguity
- 隣接定義: 辺または頂点を共有する圏域
- 行標準化（Row-standardization）
- 離島対応: KNN=1で最近隣圏域を強制的に接続

#### 構築コード例
```python
from libpysal.weights import Queen, KNN
import geopandas as gpd

gdf = gpd.read_file('secondary_areas.geojson')
w_queen = Queen.from_dataframe(gdf)
# 孤島検出
islands = [i for i in w_queen.islands]
# KNN補完
w_knn = KNN.from_dataframe(gdf, k=1)
# 統合
for island in islands:
    w_queen.neighbors[island] = w_knn.neighbors[island]
```

### 6.3 Moran's I

#### Global Moran's I
```
I = (N/S0) * Σi Σj wij(xi - x̄)(xj - x̄) / Σi(xi - x̄)²
```
- N: 観測数（335）
- wij: 空間重み行列の要素
- S0: Σi Σj wij

#### 期待値・分散
```
E[I] = -1/(N-1)
Var[I] = ... （正規近似またはPermutation test）
```

### 6.4 空間回帰モデル選択フロー

```
1. OLS回帰実施
   ↓
2. 残差のMoran's I検定
   ↓ (有意)
3. LM tests実施
   ↓
4. 以下の基準で選択:
   - LM-Lag有意 & Robust LM-Lag有意 → SLM
   - LM-Error有意 & Robust LM-Error有意 → SEM
   - 両方有意 → AIC/BIC比較
   ↓
5. 最終モデルで推定
```

## 9. リスクと対策

### 9.1 データ関連リスク

| リスク | 影響 | 対策 |
|---|---|---|
| 二次医療圏別データの欠損 | 解析N減少 | 多重代入法で補完、感度分析で確認 |
| 緑地率データの取得困難 | 解析遅延 | 都道府県単位へのフォールバック |
| NDB薬剤コードの特定困難 | データ抽出不可 | 診療行為コード（精神科専門療法）で代替 |

### 9.2 統計的リスク

| リスク | 影響 | 対策 |
|---|---|---|
| 空間的自己相関が非有意 | 空間モデル不要 | OLS結果を主とし、記述的分析を強化 |
| 緑地率と処方薬が非有意 | 帰無仮説採択 | 層別解析で部分的関連を探索 |
| 多重共線性（VIF > 10） | 係数の不安定性 | Ridge回帰、主成分分析の検討 |

### 9.3 スケジュールリスク

| リスク | 影響 | 対策 |
|---|---|---|
| データ前処理の長期化 | 解析開始遅延 | Phase 1に予備週を設定 |
| 論文執筆の遅延 | 投稿スケジュール圧迫 | 並行作業（解析中に背景執筆） |

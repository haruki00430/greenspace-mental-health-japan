### 3. 統計的検出力の再評価

#### v1.0の想定（誤り）
- N=335（二次医療圏）
- Power ≈ 0.95（効果量r=0.3）

#### v2.0の現実的評価
```
Multiple regression (Fixed model)
- Sample size: N=47（都道府県）
- Predictors: 7（緑地率＋調整変数6個）
- Effect size: r=0.3（中程度の相関）
- α=0.05（両側検定）

→ Power ≈ 0.65（やや低いが実施可能）
  r≥0.35なら Power ≈ 0.80（推奨水準）
```

**結論**: N=47でも実施可能。先行研究（NDB_XXX_slope_fracture）でも同様のN数で有意な結果を得ている。

## 📝 研究の強みと限界

### 強み
1. **オープンデータの統合**: NDB×国土数値情報×e-Statの初の統合研究
2. **空間統計の適用**: 空間的自己相関を考慮した厳密な解析
3. **政策的意義**: 都市計画×精神保健のエビデンス創出
4. **実現可能性**: 既存データ活用で4ヶ月で完遂可能

### 限界
1. **サンプルサイズ**: N=47は中規模（N=335ほど詳細ではない）
2. **生態学的誤謬**: 個人レベルの因果推論は不可
3. **時系列解析不可**: 横断研究のため因果関係の強い証拠にならない
4. **データ年度のずれ**: L03-b（2021年）とNDB（2020年度）に+1年差

### 対応策
- 限界1: 効果量が大きければ検出可能（先行研究で実績あり）
- 限界2: Discussionで明記
- 限界3: 将来的に複数年度パネルデータで検証
- 限界4: ±1年は土地利用の変化が小さく影響軽微と考察

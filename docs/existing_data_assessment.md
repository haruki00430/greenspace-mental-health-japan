## 🔍 適切性の評価

### A38二次医療圏データ

#### ✅ 適切な点
1. **年度の一致**: A38-20（令和2年、2020年）は国勢調査2020共変量と一致。**NDB第10回は令和5年度（2023年）のレセプト**（第7回が令和2年度/2020年）。
2. **データ完全性**: 全47都道府県分が存在
3. **変換済み**: GMLからGeoJSONへの変換が完了済み
4. **属性の充実**: 二次医療圏コード（A38b_003）、人口データが含まれる

#### ⚠️ 注意点
1. **本研究では都道府県レベルの解析**: 二次医療圏データを都道府県に統合（dissolve）する必要あり
2. **ポリゴンのマルチパート**: 1つの二次医療圏が複数のパーツ（離島など）に分かれている場合あり
   - 解決策: `dissolve(by='A38b_003', aggfunc='first')`で統合

#### 📊 使用方法（本研究向け）
```python
import geopandas as gpd

# Layer 2（二次医療圏）を全都道府県分読み込み
gdf_list = []
for pref_code in range(1, 48):
    path = f"A38-20_{pref_code:02d}_GML/A38-20_{pref_code:02d}_GML/A38-20_{pref_code:02d}_2.geojson"
    gdf = gpd.read_file(path)
    gdf['pref_code'] = f"{pref_code:02d}"
    gdf_list.append(gdf)

# 全国統合
gdf_national = pd.concat(gdf_list, ignore_index=True)

# 都道府県ポリゴンを作成（本研究用）
gdf_pref = gdf_national.dissolve(by='pref_code', aggfunc='first')
```

#### 結論
**✅ 本研究に使用可能（都道府県ポリゴン作成のベースデータとして最適）**

- 緑地率計算には全ファイルの処理が必要

##### ステップ2: 全メッシュを読み込み・統合
```python
import geopandas as gpd
import glob

# L03-b-21の全ZIPファイルを解凍・読み込み
gdf_list = []
for zip_path in glob.glob("data/raw/L03_b_Land_Use_2021/*.zip"):
    gdf = gpd.read_file(f"zip://{zip_path}")
    gdf_list.append(gdf)

# 全国統合
gdf_landuse = pd.concat(gdf_list, ignore_index=True)
```

##### ステップ3: 都道府県別緑地率を計算
```python
# 都道府県ポリゴンとの空間結合
gdf_landuse_with_pref = gpd.sjoin(gdf_landuse, gdf_pref, how='left', predicate='intersects')

# 緑地コード（0100, 0200, 0500, 1400）のメッシュ面積を集計
greenspace_codes = ['0100', '0200', '0500', '1400']
gdf_green = gdf_landuse_with_pref[gdf_landuse_with_pref['L03b_002'].isin(greenspace_codes)]

# 都道府県別緑地面積
greenspace_area = gdf_green.groupby('pref_code').geometry.area.sum()

# 都道府県別総面積
total_area = gdf_pref.geometry.area

# 緑地率
greenspace_ratio = (greenspace_area / total_area) * 100
```

#### 結論
**⚠️ L03-b-21（2021年）を使用すべき（既存フォルダから抽出が必要）**

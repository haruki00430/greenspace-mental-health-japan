## 2. 国土数値情報（国土交通省）

### 2.1 二次医療圏データ（A38）

#### ダウンロード先
```
https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A38.html
```

#### 対象年度
**令和2年（2020年）**: A38-20

#### ファイル形式
- GML（Geographic Markup Language）
- GeoJSON変換後使用

#### ダウンロードURL（都道府県別）
```
https://nlftp.mlit.go.jp/ksj/gml/data/A38/A38-20/A38-20_{NN}_GML.zip
```
- `{NN}`: 都道府県コード（01-47）
- 例: 北海道 → `A38-20_01_GML.zip`

#### Layer情報
- **Layer 2** (`_2.geojson`): 二次医療圏ポリゴン
- **重要属性**:
  - `A38b_003`: 二次医療圏コード（4桁、例: 0101）
  - `A38b_007`: 総人口
  - `A38b_011`: 65歳以上人口

#### 処理方法
```python
import geopandas as gpd

# 全都道府県のGeoJSONを統合
gdf = gpd.read_file('path/to/_2.geojson')
# 二次医療圏コードでdissolve（複数パーツをまとめる）
gdf_dissolved = gdf.dissolve(by='A38b_003', aggfunc='first')
```

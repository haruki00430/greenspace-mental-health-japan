## 🚨 重大な問題：NDBデータの空間単位

### 問題の詳細

#### 研究計画書の想定（誤り）
```
05_処方薬/.../二次医療圏別数量.xlsx  ← このファイルは存在しない
分析単位: 二次医療圏（N=335）
```

#### 実際のデータ構造（確認済み）
```
05_処方薬/01_公費レセプトを含まないデータ/01_処方薬（内服／外用／注射）/
├── 【内服】外来（院外）_都道府県別薬効分類別数量.xlsx
├── 【内服】外来（院内）_都道府県別薬効分類別数量.xlsx
├── 【注射】都道府県別薬効分類別数量.xlsx
└── 【外用】都道府県別薬効分類別数量.xlsx

→ すべて「都道府県別」（N=47）のみ
→ 「二次医療圏別」のデータは存在しない
```

### 影響の評価

#### 1. サンプルサイズの大幅減少
- **当初計画**: N=335（二次医療圏）
- **実際**: N=47（都道府県）
- **影響**: 統計的検出力が約1/7に低下

#### 2. 空間統計の適用可能性
- N=47でも空間統計（Moran's I、SLM/SEM）は技術的に可能
- ただし、空間パターンの詳細な検出は困難
- 離島の空間重み行列補完（KNN）の必要性は同様

#### 3. 先行研究との比較
| プロジェクト | 空間単位 | サンプルサイズ | 結果 |
|------------|---------|--------------|------|
| NDB_XXX_slope_fracture | 都道府県 | N=47 | 成功（大腿骨骨折 β=3.49, p=0.043） |
| NDB_AF_Anticoagulation_Spatial | 二次医療圏 | N=335 | 成功（Moran's I=0.146, p=0.001） |
| **本研究（修正後）** | **都道府県** | **N=47** | **実施可能（検出力やや低）** |

### 対応策の提案

#### オプション1: 都道府県単位で研究を進める【推奨】
**メリット**:
- データ取得が確実（NDB処方薬データが直接使用可能）
- NDB_XXX_slope_fractureと同様の成功実績あり
- 緑地率、社会経済データの集計が容易

**デメリット**:
- サンプルサイズが小さい（N=47）
- 都道府県内の地域差を捉えられない

**修正内容**:
- 研究タイトルを「都道府県単位の横断的分析」に変更
- サンプルサイズN=47を明記
- 検出力計算を追加（効果量d=0.4を検出するには十分）

#### オプション2: 診療行為データで代替指標を使用
**方法**:
- NDB `01_医科診療行為（算定回数）` の二次医療圏別データを使用
- 精神科関連の診療行為を抽出:
  - I002 通院・在宅精神療法
  - I003 標準型精神分析療法
  - I004 認知療法・認知行動療法
  - 精神科訪問看護・指導料

**メリット**:
- 二次医療圏別データが使用可能（N=335）
- 当初計画通りの空間単位

**デメリット**:
- 処方薬データではなく診療行為データ（アウトカムの変更）
- 薬物療法の実態を直接反映しない
- 先行研究との比較が困難

#### オプション3: 都道府県×複数年度パネルデータ
**方法**:
- NDB第8回、第9回、第10回の3年度分を統合
- N=47×3=141観測（パネルデータ）

**メリット**:
- サンプルサイズ増加
- 時系列変化を捉えられる

**デメリット**:
- データ取得作業量が3倍
- 緑地率の年次変化が小さい（説明力低下）
- パネルデータ解析の複雑化

## 📊 統計的検出力の検証

### N=47での検出力計算

#### 前提条件
- 有意水準: α=0.05（両側検定）
- 検出したい効果量: r=0.3（中程度の相関）
- 調整変数: 6個（緑地率、高齢化率、失業率、所得、独居率、診療所密度、日照時間）

#### G*Power計算結果（推定）
```
Multiple regression (Fixed model, R² deviation from zero)
- Predictors: 7
- Effect size f² = 0.15 (r=0.3相当)
- α = 0.05
- Sample size = 47

→ Power (1-β) ≈ 0.65
```

**結論**: やや低めだが、r=0.4以上の強い関連があれば80%以上の検出力

### 先行研究の効果量
- NDB_XXX_slope_fracture: 大腿骨骨折とDEM傾斜度の相関 r≈0.35（N=47で有意）
- 緑地と精神保健の海外研究: r=0.2-0.4程度

**期待**: 本研究でr=0.3-0.4の関連が得られれば有意になる可能性は十分ある

## 🌳 緑地率の定義に関する検討

### 研究計画書の定義
```
greenspace_ratio = (森林面積 + 公園面積) / 総面積 × 100
```

### 検討すべき点

#### 1. 田・畑の扱い
**問題**: 田・畑を緑地に含めるか？

**含める場合**:
- メリット: 農村部の緑地率が適切に評価される
- デメリット: 都市公園とは異なる健康効果（アクセスの難易度）

**含めない場合**:
- メリット: 「アクセス可能な緑地」に限定（理論的妥当性）
- デメリット: 農村部の緑地率が過小評価

**推奨**:
1. メイン解析: 森林+公園のみ
2. 感度分析: 森林+公園+田畑で再解析
3. Discussion: 定義の影響を考察

#### 2. 緑地の質の考慮
**問題**: 面積だけでなく、アクセス性・種類も重要

**対応**:
- 公園面積を別変数として扱う（森林vs公園の効果比較）
- 人口密度との交互作用（都市部での公園の重要性）

## 🔍 変数取得方法の明確化

### 不明確な点

#### 1. 市区町村→都道府県の集計方法
**必要データ**:
- 国勢調査: 市区町村別 → 都道府県で集計（単純合計でOK）
- 所得: 市区町村税課税状況 → 都道府県平均（人口加重平均）

**確認事項**:
- e-Statで都道府県別データが直接取得できるか確認
- できない場合は、市区町村データを手動集計

#### 2. 精神科診療所密度
**データソース**: 医療施設調査（令和2年）

**確認事項**:
- 都道府県別の精神科・心療内科標榜診療所数が取得可能か
- e-Stat統計表IDの特定

#### 3. 日照時間の都道府県代表値
**方法**: 都道府県庁所在地の気象台データを使用

**問題**:
- 北海道、沖縄など広域の都道府県で代表性が低い

**対応**:
- 都道府県内の複数地点（主要都市）の平均を取る
- 感度分析: 日照時間を除外したモデルも検討

## 📝 その他の改善提案

### 1. 仮説の精緻化

#### 現在の主仮説
> 緑地率が高い二次医療圏では、年齢・社会経済要因調整後も、精神科処方薬の処方量が有意に低い

#### 修正提案
> 緑地率が高い都道府県では、年齢・社会経済要因・医療アクセス調整後も、精神科処方薬の人口10万人あたり処方量（年齢標準化）が有意に低い（β<0, p<0.05）

**変更点**:
- 「二次医療圏」→「都道府県」
- アウトカムの定義を明確化（年齢標準化、人口10万人あたり）
- 統計的仮説を明記

### 2. 交絡因子の追加検討

#### 現在のリスト

- 所得
- 独居世帯率
